# omxlooper

Loop a video on the RaspberryPi.  Once started, it can be controlled through
a ZeroMQ socket to play another file, pause or stop.

Based on the `hello_video` sample code.
Only h264 videos and no audio.

## Protocol

- open file and play
- increase speed
- decrease speed
- play
- pause
- stop
- shutdown
